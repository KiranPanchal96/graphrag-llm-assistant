# graphrag-llm-assistant

RAG / GraphRAG side project:
- src: contains source code files
- data: empty as private data files have been stripped out - text-based Word documents were used
- tests: includes unit testing scripts
- requirements.txt: list of packages used
- Dockerfiles

Tech stack:
- LangChain (orchestration)
- networkx and Neo4J (graph database)
- Pydantic (data validation)
- fastapi (api)
- Gradio (UI)
- pytest (unit testing)
- langfuse (prompt management and evals)
- Ollama (DeepSeek open-source model) and OpenAI API (GPT-4 models)
- Docker (deployment)
